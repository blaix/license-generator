A permissive license that also provides an express grant of patent rights from contributors to users.

*  State Changes

### [GPL](http://choosealicense.com/licenses/gpl-2.0/)

GPL is the most widely used free software license and has a strong copyleft requirement. When distributing derived works, the source code of the work must be made available under the same license. There are multiple variants of the GPL, each with different requirements.

* [GNU Affero GPL v3.0](http://choosealicense.com/licenses/#agpl-v3)
* [GNU GPL v2.0](http://choosealicense.com/licenses/#gpl-v2)
* [GNU GPL v3.0](http://choosealicense.com/licenses/#gpl-v3)

Required

*  Disclose Source
*  License and copyright notice
*  State Changes

Permitted

*  Commercial Use
*  Distribution
*  Modification
*  Patent Grant
*  Private Use

Forbidden

*  Hold Liable
*  Sublicensing

[View full GNU General Public License v3.0 license »](http://choosealicense.com/licenses/gpl-3.0/)

### [LGPL](http://choosealicense.com/licenses/lgpl-2.1/)

* [GNU LGPL v2.1](http://choosealicense.com/licenses/#lgpl-v2_1)
* [GNU LGPL v3.0](http://choosealicense.com/licenses/#lgpl-v3)

### [Public Domain Dedication](http://choosealicense.com/licenses/unlicense/)

Because copyright is automatic in most countries,[the Unlicense](http://unlicense.org/) is a template to waive copyright interest in software you've written and dedicate it to the public domain. Use the Unlicense to opt out of copyright entirely. It also includes the no-warranty statement from the MIT/X11 license.

* [CC0 1.0 Universal](http://choosealicense.com/licenses/#cc0)
* [The Unlicense](http://choosealicense.com/licenses/#unlicense)
